# io_scene_owm 3.x [![Discord](https://img.shields.io/discord/346445737367699456.svg?label=&logo=discord&logoColor=ffffff&color=7389D8&labelColor=6A7EC2)](https://discord.gg/XM93ZdB)

Blender importer add-on for Overtools/DataTool file formats.
**Blender 4.2+ is required.**

## Automatic Installation

1. Visit https://extensions.blender.org/add-ons/io-scene-owm/
2. "Get Add-on"
3. Done.

## Manual Installation

1. Download the latest release from the [releases tab](https://github.com/overtools/io_scene_owm/releases).
2. In Blender, go to the `Edit->Preferences->Add-ons` menu.
3. Click `top right arrow -> Install from Disk`. **Be sure to remove any existing version**
4. Done.
